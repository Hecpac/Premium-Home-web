# PROJECT BRIEF: Premium Home Design - Infraestructura Digital High-Ticket

## 1. Visión del Proyecto
Construir una plataforma de conversión de alto rendimiento para una constructora de casas de lujo en Dallas/Las Colinas. El sitio no es un catálogo; es un motor de "Confianza Operacional" [2]. Debe demostrar capacidad técnica superior y eliminar la ansiedad del comprador mediante claridad radical.

## 2. Objetivos de Negocio (KPIs)
*   **Velocidad:** LCP < 2.5s y cero Layout Shifts (CLS < 0.1) para maximizar retención móvil [3].
*   **Conversión:** Incrementar leads cualificados (presupuesto >$500k) mediante formularios multi-paso [4].
*   **Autoridad:** Posicionamiento como entidad local dominante en Google (GEO - Generative Engine Optimization) [5].

## 3. Público Objetivo (Target Audience)
*   **Perfil:** High Net Worth Individuals (HNWI), familias trasladándose a Dallas, inversores inmobiliarios.
*   **Dolores:** Miedo a estafas, retrasos en construcción, costos ocultos.
*   **Deseos:** Exclusividad, transparencia radical, tecnología de construcción avanzada.

## 4. Stack Tecnológico (No Negociable) [6, 7]
*   **Framework:** Next.js 14+ (App Router) para renderizado híbrido y SEO.
*   **UI System:** Shadcn UI + Tailwind CSS v4 (Componentes puros, sin dependencias ocultas).
*   **Motion:** Framer Motion para micro-interacciones físicas (spring physics).
*   **3D/Visuals:** Three.js + TSL (WebGPU) para shaders de alto rendimiento [8].

## 5. Estructura Narrativa (StoryBrand Script) [9, 10]
*   **Héroe:** El cliente que busca construir su legado.
*   **Guía:** Premium Home Design (Expertos en zonificación de Las Colinas).
*   **Plan:** Diseño 3D -> Construcción Transparente -> Entrega Llave en Mano.
*   **Villano:** El caos de la construcción y los contratistas opacos.
